wx.GraphicsBitmap¶

Represents a bitmap.

The objects of this class are not created directly but only via wx.GraphicsContext or wx.GraphicsRenderer CreateBitmap(), CreateBitmapFromImage() or CreateSubBitmap() methods. They can subsequently be used with wx.GraphicsContext.DrawBitmap . The only other operation is testing for the bitmap validity which can be performed using IsNull method inherited from the base class.

ConvertToImage(self)¶

Return the contents of this bitmap as wx.Image.

Using this method is more efficient than converting wx.GraphicsBitmap to wx.Bitmap first and then to wx.Image and can be useful if, for example, you want to save wx.GraphicsBitmap as a disk file in a format not directly supported by wx.Bitmap.

Invalid image is returned if the bitmap is invalid.

Return type

wx.Image

New in version 2.9.3.
