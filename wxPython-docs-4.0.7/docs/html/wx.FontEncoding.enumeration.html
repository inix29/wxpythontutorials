wx.FontEncoding¶

The FontEncoding enumeration provides the following values:

Description

Value

wx.FONTENCODING_SYSTEM

Default system encoding.

wx.FONTENCODING_DEFAULT

Default application encoding.

Cyrillic.

wx.FONTENCODING_ISO8859_6

Arabic.
