class AuiManager_DCP(AuiManager)¶

Update(self)¶

This method is called after any number of changes are made to any of the managed panes. Update must be invoked after AuiManager.AddPane or AuiManager.InsertPane are called in order to “realize” or “commit” the changes.

In addition, any number of changes may be made to AuiManager structures (retrieved with AuiManager.GetPane), but to realize the changes, Update must be called. This construction allows pane flicker to be avoided by updating the whole layout at one time.
