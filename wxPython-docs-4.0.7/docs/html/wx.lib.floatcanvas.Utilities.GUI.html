wx.lib.floatcanvas.Utilities.GUI¶

Part of the floatcanvas.Utilities package.

This module contains assorted GUI-related utilities that can be used with FloatCanvas

So far, they are:

RubberBandBox: used to draw a RubberBand Box on the screen
