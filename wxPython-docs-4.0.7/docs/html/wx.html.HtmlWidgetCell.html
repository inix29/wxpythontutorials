wx.html.HtmlWidgetCell¶

wx.html.HtmlWidgetCell is a class that provides a connection between HTML cells and widgets (an object derived from wx.Window).

You can use it to display things like forms, input boxes etc. in an HTML window.

wx.html.HtmlWidgetCell takes care of resizing and moving window.

  • wnd (wx.Window) – Connected window. It is parent window must be the wx.html.HtmlWindow object within which it is displayed!

  • w (int) – Floating width. If non-zero width of wnd window is adjusted so that it is always w percents of parent container’s width. (For example w = 100 means that the window will always have same width as parent container).
