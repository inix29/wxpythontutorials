wx.lib.docview.DocChildFrame¶

The DocChildFrame class provides a default frame for displaying documents on separate windows. This class can only be used for SDI (not MDI) child frames.

The class is part of the document/view framework and cooperates with the View, Document, DocManager and DocTemplate classes.
