wx.xml.XmlAttribute¶

Represents a node attribute.

Example: in <img src="hello.gif" id="3"/> , src is an attribute with value hello.gif and id is an attribute with value 3 .

See also

wx.xml.XmlDocument, wx.xml.XmlNode
