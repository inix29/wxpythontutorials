wx.GBPosition¶

This class represents the position of an item in a virtual grid of rows and columns managed by a wx.GridBagSizer.

GetIM(self)¶

Returns an immutable representation of the wx.GBPosition object, based on namedtuple.

This new object is hashable and can be used as a dictionary key, be added to sets, etc. It can be converted back into a real wx.GBPosition with a simple statement like this: obj = wx.GBPosition(imObj).
