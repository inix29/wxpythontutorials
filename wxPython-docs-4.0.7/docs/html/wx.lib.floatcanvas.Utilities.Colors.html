wx.lib.floatcanvas.Utilities.Colors¶

Colors.py

Assorted stuff for Colors support. At the moment, only a few color sets.

Many of these are from: http://geography.uoregon.edu/datagraphics/color_scales.htm

They may have been modified some

CategoricalColor1: A list of colors that are distict. BlueToRed11: 11 colors from blue to red
