wx.GraphicsFont¶

A wx.GraphicsFont is a native representation of a font.

The contents are specific and private to the respective renderer. Instances are ref counted and can therefore be assigned as usual. The only way to get a valid instance is via wx.GraphicsContext.CreateFont or wx.GraphicsRenderer.CreateFont .
